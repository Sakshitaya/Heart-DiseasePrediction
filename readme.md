# ❤️ Heart Disease Prediction

## 📌 Overview
Heart Disease Prediction is a web-based application that uses machine learning to predict the risk of heart disease based on user inputs such as age, gender, chest pain type, cholesterol level, etc. The system helps users understand their health condition and generates a basic prediction report.

---

## 🚀 Features
- User-friendly web interface for inputting health data  
- Predicts heart disease risk using trained ML models  
- Supports multiple algorithms for better accuracy  
- Generates a report for users  
- Simple and responsive UI  

---

## ⚙️ Tech Stack
- **Languages:** Python, JavaScript, HTML, CSS  
- **Framework:** Flask  
- **Algorithms:** Logistic Regression, SVM, Decision Tree, Random Forest, KNN  
- **Libraries:** NumPy, Pandas, Matplotlib  
- **Tools:** VS Code, Jupyter Notebook
